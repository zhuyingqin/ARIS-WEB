## ‚ú® What is ARIS-Code?

**ARIS-Code** (*Auto Research in Sleep*) is a terminal-based AI research assistant built for academic researchers. Its core philosophy:

- ūü§Ė **Executor**: The primary LLM ‚ÄĒ writes code, surveys literature, drafts papers, plans experiments
- ūüĒć **Reviewer**: An independent LLM that adversarially critiques the Executor's output via the `LlmReview` tool
- ūüĒĄ **Iterate**: Executor writes ‚Üí Reviewer critiques ‚Üí Executor revises ‚Üí loop until quality converges

With **42 bundled research skills**, ARIS covers the full pipeline from idea discovery to paper submission.

```
User input
    ‚Üď
```

### 4. ūüĆź Language Preference

Your chosen language (CN/EN) is injected into the system prompt so ARIS always responds in your preferred language ‚ÄĒ no per-message configuration needed.

### 5. ūüõ°ÔłŹ Anti-Hallucination Design

The system prompt explicitly informs the model of its exact identity (ARIS-Code), preventing role confusion in multi-agent scenarios where the Executor and Reviewer are different models from different providers.
